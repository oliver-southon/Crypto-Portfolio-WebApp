to create sqlalchemy database:

>1. pip install flask_sqlalchemy

>2. app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///newDatabaseName.db'
    app.config['SECRET_KEY'] = str(os.environ.get('DB_DK'))

>3. db = SQLAlchemy(app)

>4. Create Models, eg:

        class Holding(db.Model):
            holding_id = db.Column(db.Integer, primary_key=True)
            symbol = db.Column(db.String(10), nullable=False)
            entry_price = db.Column(db.Float, nullable=False)
            entry_amt = db.Column(db.Float, nullable=False)
            date = db.Column(db.DateTime)

>5. run python shell
>6. 'from main import db
>7. db/create_all()
